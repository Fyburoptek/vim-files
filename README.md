# VIm Dot-Files

## Installation
Make sure your `.vimrc` file is backed up before copying over with this one.

    1. Clone repository using `git@github.com:Fyburoptek/vim-files.git`
    2. Copy `.vim` folder to home folder
    3. Copy `.vimrc` to home folder
